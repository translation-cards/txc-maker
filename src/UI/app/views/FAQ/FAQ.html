What is TXCMaker?

TXCMaker is a tool for creating .txc files, which are files that can be imported into the Translation Cards mobile app. These files contain the audio recordings and information for displaying translations. For more information on the Translation Cards project check out our public site at translation-cards.com

Who should use TXC Maker?

TXC Maker is a tool for anyone who is interested in creating a translation deck for the Translation Cards mobile application. It is primarily aimed at field workers for use in reception desks in the context of the refugee crisis, however is not limited to these scenarios.

What should I have ready before using TXC Maker?

You should have two documents ready before using the TXC Maker:

  • An excel template with all the deck content. You can copy this excel template and fill out a line for every translation you'd like to make.
    • Under the Language heading you should fill out the iso code for the language you'd like to use not the language itself. Not sure what the iso code is for your language? Check here.
      For example if the language for my translation is Spanish, I would fill out 'es' under the language column for that translation.
    • Write the english phrase to be translated in the 'label' column
    • The translation column should hold the translated phrase in the language you filled out
    • Finally, put the name of the translation audio recording under the filename column
    • Here you can view an example of a filled out sheet that is in use for the default deck in the application.
    • Make sure you save your spreadsheet or import it into google drive
  • A folder in your Google drive with all your audio files containing the recorded translations for your deck.

Using TXC Maker

  1. Navigate to deck import to get to the deck import page.
  2. Input the name of your deck, and the publisher name (organization name or author of deck).
  3. Under the document ID input the the portion after the last / of the URL containing your deck spreadsheet in google drive.
    For example if the URL with your spreadsheet looks like this 'https://drive.google.com/drive/u/1/folders/0BzScrGiXtpD_dnBRZ1BOWms5dWM', you just need to input '0BzScrGiXtpD_dnBRZ1BOWms5dWM' under document ID.
  4. Similarly in the audio directory input the portion after the last / of the Google Drive URL containing all your recorded audio translation files.
  5. Select the license and whether you want to lock the deck (which means users will NOT be allowed to add, change, or remove translations from your deck).
  6. Finally, submit and wait for the deck maker to compose your deck into a TXC. It will catch any possible mistakes and show you on the following screen.
  7. If any errors are caught, they must be resolved before you can publish the deck.
